### 特征

> 用 C++ 和 Qt 编写的跨平台 UI

  * 基于吸血鬼的现代黑暗主题

#### 团队服务器

> 用 Go 语言编写

  * 多人游戏

  * 有效载荷生成（exe/shellcode/dll）

  * HTTP/HTTPS 侦听器

  * 可定制的 C2 配置文件

  * 外部C2

> Havoc 的旗舰代理，用 C 和 ASM 编写

  * 通过Ekko或FOLIAGE进行睡眠混淆

  * x64 返回地址欺骗

  * Nt* API 的间接系统调用

  * 中小企业支持

  * 令牌库

  * 各种内置的后期开发命令

![](https://gitee.com/fuli009/images/raw/master/public/20230109095003.png)

  * External C2

  * Custom Agent Support

    * Talon

  * Python API

  * Modules
